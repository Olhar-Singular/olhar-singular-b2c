# Orientador Digital B2C

Projeto B2C do Orientador Digital. Stack técnica idêntica ao projeto B2B `orientador-digital`.

## Comandos

Ver todos os comandos disponíveis:

```bash
make help
```

Subir ambiente local completo:

```bash
cp .env.example .env   # preencher VITE_SUPABASE_* após make sb-status
make start             # Supabase local + container app (porta 8080)
```
